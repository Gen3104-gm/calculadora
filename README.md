# Calculadora

Una calculadora web interactiva creada con HTML, CSS y JavaScript.

## Características

- Operaciones básicas: suma, resta, multiplicación, división
- Interfaz intuitiva y responsive
- Diseño moderno

## Tecnologías

- HTML5
- CSS3
- JavaScript

## Cómo usar

Simplemente abre el archivo `index.html` en tu navegador.
